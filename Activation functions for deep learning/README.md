# Activation Functions for Deep Learning

This folder contains the code and Jupyter notebooks for the **Activation functions for deep learning** topic.

The notebook serves as a comprehensive, beginner-friendly guide taking you through the fascinating world of activation functions—the components that give neural networks their power to learn complex patterns.

### Covered Topics:
- **Why Activations Matter:** Linear vs non-linear, universal approximation.
- **Classic Activations:** Sigmoid, Tanh, Step function.
- **ReLU Revolution:** ReLU and the dying ReLU problem.
- **ReLU Variants:** LeakyReLU, PReLU, ELU, SELU.
- **Modern Activations:** GELU, Swish, Mish, SiLU.
- **Output Activations:** Softmax, Sigmoid for outputs.
- **Practical Guide:** Choosing the right activation function for your task.
- **Hands-on Comparison:** Training networks with different activations.

**Note:** Video tutorials explaining the code are available in this folder to help you follow along effectively!
