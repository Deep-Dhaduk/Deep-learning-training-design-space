# Convolution Neural Networks

This folder contains the code and Jupyter notebooks for the **Convolution Neural Networks** topic.

The notebook explores the fundamental concepts of Convolutional Neural Networks (CNNs) from first principles. It provides a visual journey into computer vision deep learning.

### Covered Topics:
- **The Convolution Operation:** Kernels, feature maps, edge detection.
- **Building CNN Layers:** `Conv2d`, padding, stride, channels.
- **Pooling & Downsampling:** `MaxPool`, `AvgPool`, spatial reduction.
- **CNN Architectures:** Patterns from LeNet and AlexNet.
- **Training CNNs:** Data augmentation, batch normalization.
- **Visualizing What CNNs Learn:** Feature maps, filters, Grad-CAM.
- **Complete CIFAR-10 Classifier:** End-to-end implementation.

**Note:** Video tutorials explaining the code are available in this folder to help you follow along effectively!
