# Metrics for classification - fundamentals

This folder contains the code and Jupyter notebooks for the **Metrics for classification - fundamentals** topic.

The notebook provides a complete guide to understanding and evaluating classification models in machine learning and deep learning. 

### Covered Topics:
- **What is Classification?** Understanding the task and its real-world applications.
- **Confusion Matrix:** Building the foundation of all classification metrics (True Positives, False Positives, etc.).
- **Key Metrics:** Accuracy, Precision, Recall (Sensitivity), and F1-Score.
- **ROC and AUC:** Visualizing and quantifying classifier performance across thresholds.
- **Multi-class Metrics:** Macro, Micro, and Weighted averaging techniques.
- **Imbalanced Datasets:** Handling scenarios where classes are not represented equally.
