# Modern CNN Architectures and Transfer Learning

This folder contains the code and Jupyter notebooks for the **Modern CNN architectures and Transfer Learning** topic.

The notebook explores advanced Convolutional Neural Network structures and how to effectively leverage pre-trained models across different computer vision tasks.

### Covered Topics:
- **Evolution of CNNs:** Journey from LeNet to AlexNet, VGG, ResNet, and beyond.
- **Architectural Innovations:** Skip connections, inception modules, and depthwise separable convolutions.
- **VGG Networks:** Building deeper networks with smaller kernels.
- **ResNets:** Overcoming the vanishing gradient problem with residual blocks.
- **Transfer Learning:** Leveraging pre-trained models on new datasets.
- **Fine-Tuning vs. Feature Extraction:** Strategies for adapting existing models effectively.
- **Practical Implementation:** Hands-on examples of applying transfer learning to custom classification problems.

**Note:** Video tutorials explaining the code are available in this folder to help you follow along effectively!
