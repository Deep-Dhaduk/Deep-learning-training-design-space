# Optimizers for Deep Learning

This folder contains the code and Jupyter notebooks for the **Optimizers for deep learning** topic.

The notebook serves as a comprehensive, beginner-friendly guide exploring the mechanics of optimization algorithms—the driving engines behind everything neural networks learn.

### Covered Topics:
- **Optimization basics:** Understanding loss landscapes, finding the minimum.
- **Gradient Descent:** The fundamental update rule and variations (Batch, Mini-batch, Stochastic).
- **Momentum:** Adding velocity to gradients to escape local minima and avoid oscillation.
- **Adaptive Learning Rates:** Algorithms that automatically tune the learning rate.
- **AdaGrad & RMSprop:** Scaling the learning rate based on historical gradients.
- **optimizers like Adam & RMSprop:** Advanced techniques combining momentum and adaptive learning rates.
- **Scheduler and Learning Rate decay:** Adjusting learning rate dynamically during training.
- **Visualizing loss:** Demonstrations comparing optimizer performance on complex loss surfaces.
