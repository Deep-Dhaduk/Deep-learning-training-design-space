# Deep Learning Training Design Space

Welcome to the Deep Learning Training Design Space repository! This repository contains various modules and tutorials covering fundamental concepts, architectures, and optimization techniques in deep learning.

## Modules Included:
- **Activation functions for deep learning**
- **Convolution Neural Networks**
- **Hyperparameter tuning for deep learning**
- **Metrics for classification - fundamentals**
- **Modern CNN architectures and Transfer Learning**
- **Optimizers for deep learning**

**Note:** Video tutorials explaining the code are available in their respective folders! Each topic has its own folder containing the relevant Jupyter notebooks, code, and video explanations.
